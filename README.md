# Proyecto Módulo 1 – Fundamentos (Streamlit)

## Descripción

Aplicación desarrollada en Streamlit que integra los conceptos fundamentales del Módulo 1 de la Especialización en Python for Analytics.  

La aplicación incluye:

- Variables y condicionales
- Listas y diccionarios
- Funciones y programación funcional 
- Programación orientada a objetos (POO)

Cada ejercicio está organizado como un módulo dentro de una aplicación interactiva construida con Streamlit.

---

## Objetivo del Proyecto

Desarrollar una aplicación web interactiva que permita aplicar los fundamentos de programación en Python mediante el registro y análisis de actividades financieras.

---

## Estructura que manejé en el proyeto repositorio
|---- app.py
|---- requirements.txt
|---- README.md

## Explicación de la estructura
- **app.py** → Archivo principal de la aplicación.
- **requirements.txt** → Lista de dependencias necesarias.
- **README.md** → Documentación del proyecto.

## Funcionalidades por Módulo

### Home
- Presentación del proyecto.
- Información del estudiante.
- Descripción del objetivo.
- Tecnologías utilizadas.

### Ejercicio 1 – Variables y Condicionales
Permite ingresar un presupuesto y un gasto para verificar si se cumple el presupuesto.

### Ejercicio 2 – Listas y Diccionarios
Registro de actividades financieras utilizando una lista de diccionarios.

### Ejercicio 3 – Funciones y Programación Funcional
Cálculo del retorno esperado usando una función aplicada con `map` y `lambda`.

Fórmula del retorno:
Retorno = presupuesto × tasa × meses


### Ejercicio 4 – Programación Orientada a Objetos
Implementación de la clase `Actividad` con:
- Atributos: nombre, tipo, presupuesto, gasto_real
- Métodos:
  - `esta_en_presupuesto()`
  - `mostrar_info()`
